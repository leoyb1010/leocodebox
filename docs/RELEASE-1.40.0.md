# leocodebox 1.40.0

## 这一版你多了什么

- **秒开**：关窗改为收进托盘/Dock，服务保温运行，再次打开即刻回到工作区；「本地设置」新增「退出后保温」开关，完全退出后下次启动可直接接管已运行的服务（版本不匹配自动杀旧起新）。
- **原地站回**：启动直接回到上次会话，草稿仍在输入框里；上次会话已删除时自动回落到常规首页。
- **完成召回**：Dock 角标实时显示运行中会话数；Agent 长任务（>30 秒）完成或失败时发原生通知，点击直达该会话；窗口聚焦时不打扰。
- **接口一条线**：状态栏显示当前会话走的 Leoapi 节点昵称与延迟；⌘K 可直接切换/测速节点（二次确认，事务化，失败自动回滚）。
- **Leoapi 三卡叙事**：接口列表按「本机原配置 / 今天在用 / 备选节点」分区，备选按健康度与延迟排序；端点支持昵称（如「家里光猫」）；测速结果带横向延迟条带；健康徽章显示「N 分钟前检测」。
- **跨 Agent 交接**：⌘K「交接给另一个 Agent」把最近对话拼成可编辑的接力前言，切到目标 Agent 后由你确认发送。
- **搜索更全**：会话全文搜索新增覆盖 OpenCode，搜索结果标注本次覆盖了哪些 Agent。
- **干净机器友好**：空侧栏直接给出「安装本机 Agent」按钮；启动屏三步进度灯反映真实阶段。
- **一键诊断**：设置 → Agents 可导出脱敏诊断 JSON（密钥打码、家目录折叠为 ~）。

## 你几乎感觉不到但更稳的

- 启动关键路径上的 login shell 探测改为异步并缓存，重 zshrc 不再冻结界面最长 10 秒。
- 移除约 1071 行不可达的云环境遗留代码（主进程瘦身 43%），通知管线正式接通本地服务器。
- API Key 改为 SHA-256 哈希落盘（存量自动迁移）；密钥不再允许经 URL 参数传递；令牌比较统一为常量时间。
- 安装包内不再捆绑 Codex 兜底二进制（约 300MB）：未安装 codex CLI 的用户首次使用时自动下载并缓存，DMG 体积约 -27%。
- Esc 在 Leoapi 页内也能关闭弹窗；破坏性确认全部改为应用内样式化对话框。

## 刻意没做的

- 不开启 asar（签名链路 A/B 测试通过前维持现状,见 docs/SIGNING.md）。
- Cursor 历史会话暂不进入全文搜索（其分散二进制存储需要独立设计），搜索界面已如实标注覆盖范围。
- Leoapi provider 密钥维持 0600 本地文件：加密密钥与数据同盘属于安全剧场，如需更强保护将另行走系统钥匙串。
