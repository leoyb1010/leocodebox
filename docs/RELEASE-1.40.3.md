# leocodebox 1.40.3

## 这一版你多了什么

- **编辑已启用的接口立即生效**：修改当前正在使用的 Leoapi 接口（地址/密钥/模型/映射）后，配置文件即刻重写并对新会话生效，不再需要手动再点一次「启用」——此前编辑只进存储，真实配置悄悄停留在旧值。
- **切换后聊天框模型自动跟上**：启用/重应用接口时，聊天输入框记住的模型（每条消息都会显式携带）自动重置为新接口的模型，Claude/Codex/OpenCode 三个目标全覆盖——此前仅 OpenCode 生效，其余目标"换了接口，聊天还在发旧模型名"。
- **复制不再报错**：应用内所有「复制」按钮修复 Write permission denied（Electron 权限白名单补齐 clipboard-sanitized-write），并附选区复制兜底。
- **模型列表可滚动**：接口编辑与测速里的模型选择升级为「聚焦即展开全部模型、可滚动、输入过滤」的下拉，不再只能盲打搜索。
