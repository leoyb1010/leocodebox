# leocodebox 1.39.1

## 修复

- 本机智能体控制完整显示 Claude Code、Codex、OpenCode、Cursor、Gemini CLI、Hermes Agent 与 Grok Build。
- 所有已安装智能体统一显示“检查并更新”或“立即更新”，并使用对应安装来源或官方更新入口。
- 新建会话运行期间自动补拉首批消息，不再停在“正在加载会话消息”并要求二次点击。
- 工作区列表默认隐藏未收藏的零会话目录；收藏项目始终保留。
- Leoapi 弹窗、目标标签和配置卡恢复品牌图标与名称。
- 项目统计、网页终端两个兼容插件显示中文名称和说明。

## 免凭据热更新

- 新增公开签名资产仓库 `leoyb1010/leocodebox-updates`，仅发布 DMG、ZIP 和更新元数据，不公开源码。
- 1.39.1 及以后默认使用该公开 feed，无需 GitHub Token。
- 1.39.0 客户端尚未内置公开 feed 地址，需要手动安装本版一次；之后可直接应用内更新。

## 插件审计

当前兼容协议下可验证的内置插件为“项目统计”和“网页终端”，两者均已安装并可启停。未发现另一个有明确来源、维护状态和权限说明的官方插件仓库，因此没有自动安装不可信第三方代码。
