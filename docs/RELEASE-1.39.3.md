# leocodebox 1.39.3

本次版本完成第一阶段品牌视觉融合，并修复退出后被后台重新拉起的问题。

## 修复

- Browser MCP 使用 Electron 可执行文件时强制进入无界面的 Node 模式，Claude、Codex 或 Cursor 重连 MCP 不再启动 leocodebox 窗口。
- 应用启动时自动修复既有的 `leocodebox-browser` MCP 配置，已安装设备无需手动删除旧配置。
- 关闭“退出时自动安装更新”，只有明确点击“重启并安装”才允许更新器重启应用。
- 普通退出继续等待本地 38473 服务完成关闭。

## 视觉融合

- 启动等待页接入浅色、深色品牌场景，并保留日志诊断能力。
- 首次配置引导接入本地工作台与 CLI 自动发现视觉图。
- README 接入真实产品品牌主视觉。
- DMG 构建接入定制背景、应用与 Applications 图标布局。
- 收录完整生图源文件和前端交付文件。

## 资产质量说明

部分标称透明的空状态 PNG/WebP 实际将棋盘格烘焙进了像素且没有 Alpha 通道。本版不把这些图片强行放入高频操作界面，避免浅深色主题出现方格底；源文件完整保留，待透明底修正后可继续接入。
