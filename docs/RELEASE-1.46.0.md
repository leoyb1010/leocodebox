# leocodebox 1.46.0

「智能体档案」——把 leocodebox 变成你的个人智能体集合。常用的智能体设定存成命名档案,一键启动预配置好的新对话。全程门禁全绿 + 对抗式审查(修 5 个确认缺陷)。

## 多了什么

**智能体档案 Hub(设置 → 智能体档案)**
- 每个档案固定一套「CLI + 模型 + 努力度 + 权限模式 + 开场提示词 + 备注」。比如「代码审查员 = Claude / 默认模型 / high / 计划模式 / 开场:审查当前分支」,或「快速问答 = Cursor / 自动」。
- 卡片一键**启动**:自动切到该档案的 CLI、套用模型/努力度/权限模式、把开场提示词填进新对话的输入框,并打开一个全新会话。你审一眼再发,或直接发。
- 卡片支持编辑、复制、删除;弹窗编辑器带表情图标选择,动效克制丝滑。
- **JSON 导入/导出**:档案库可携带、可备份、可分享。

**刻意的边界(不臃肿、不越权)**
- 启动只复用对话框已有的偏好事件总线来套用设定,**不触碰各 CLI 的 MCP / 技能环境级配置**——因为它们是按作用域(用户/项目)落盘的环境配置,没有「单次对话级」注入点。档案绝不为了炫技去静默改你的全局配置。MCP / 技能仍在各自的一等 Tab 里由你显式管理。

## 更稳的

- 对抗式审查修掉 5 个确认缺陷:
  - **[中] 导入接口无上限**:超大导入会用同步事务塞爆数据库、阻塞事件循环。加 500 条上限(413)+ 逐元素校验。
  - **[中] 服务端错误显示「[object Object]」**:错误信封是对象而客户端当字符串读。两处请求路径统一解包 `error→message`,**全 App** 的服务端错误提示都变清晰(纯字符串错误零回归)。
  - **[低] 超长字段静默截断**:创建/更新超长内容改为 400 报错,不再无声裁掉数据;截断按码点切,不产生半个 emoji。
  - **[低] 导入非法元素变垃圾档案**:非对象元素改为 400 拒绝,不再插入默认垃圾行。
  - **[低] 启动时空模型未回默认**:兑现「留空用默认」的承诺,不再继承上次残留模型。
