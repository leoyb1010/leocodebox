# leocodebox 1.39.2

## 严重问题修复

- 修复新建会话已经运行，但消息区一直停在“正在加载会话消息”的问题。
- 修复加载结束后错误显示“继续您的对话”，必须重新点击会话列表才能看到消息的问题。
- 当前会话的实时消息写入 Store 后会立即参与渲染，不再被旧的空数组缓存阻断。

## 回归验证

- 新增新会话首条实时消息回归测试，覆盖“历史接口仍在等待、实时消息已经到达”的竞态。
- 反向恢复旧缓存后，该测试会稳定复现 `期望 1 条消息，实际 0 条`，证明测试覆盖了本次故障根因。
- 继续使用 1.39.1 建立的公开签名更新源，应用内升级不需要 GitHub Token。
